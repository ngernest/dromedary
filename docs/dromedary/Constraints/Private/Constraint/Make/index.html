The module Type provides the concrete representation of types (using constraint type variables) in constraints.

It is the free monad of the functor Type_former.t.

History: This representation was initially used in constraints t, however, the refactor for "Sharing" now uses Shallow_type.t. We however, still use Type for a rich interface.

'a t is a constraint with value type 'a.

In the meta-theory, the constraint language has a defined type system.

In our implementation, we use GADTs to implement the type system, where the type parameter 'a denotes the type of the constraint.

type _ t =
  1. | True : Base.unit t
    (*

    true

    *)
  2. | Return : 'a -> 'a t
    (*

    return a

    *)
  3. | Conj : 'a t * 'b t -> ('a * 'b) t
    (*

    C₁ && C₂

    *)
  4. | Eq : variable * variable -> Base.unit t
    (*

    ɑ₁ = ɑ₂

    *)
  5. | Exist : existential_context * 'a t -> 'a t
    (*

    exists Θ. C

    *)
  6. | Forall : universal_context * 'a t -> 'a t
    (*

    forall Λ. C

    *)
  7. | Instance : Algebra.Term_var.t * variable -> Algebra.Decoded.Type.t Base.list t
    (*

    x <= ɑ

    *)
  8. | Def : binding Base.list * 'a t -> 'a t
    (*

    def x1 : t1 and ... and xn : tn in C

    *)
  9. | Let : 'a let_binding Base.list * 'b t -> ('a term_let_binding Base.list * 'b) t
    (*

    let Γ in C

    *)
  10. | Let_rec : 'a let_rec_binding Base.list * 'b t -> ('a term_let_rec_binding Base.list * 'b) t
    (*

    let rec Γ in C

    *)
  11. | Map : 'a t * ('a -> 'b) -> 'b t
    (*

    map C f.

    *)
  12. | Decode : variable -> Algebra.Decoded.Type.t t
    (*

    decode ɑ

    *)
  13. | Implication : equations * 'a t -> 'a t
    (*

    E => C

    *)
