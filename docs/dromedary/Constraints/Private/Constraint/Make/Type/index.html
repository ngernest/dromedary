Module Make.Type

type t =
  1. | Var of variable
  2. | Former of t Type_former.t
  3. | Row_cons of Label.t * t * t
  4. | Row_uniform of t
  5. | Mu of variable * t
  6. | Let of variable * t * t

t represents the type defined by the grammar: t ::= ɑ | (t, .., t) F

val sexp_of_t : t -> Sexplib0.Sexp.t
val var : variable -> t

var 'a is the representation of the type variable 'a as the type t.

val former : t Type_former.t -> t

former f is the representation of the concrete type former f in type t.

val mu : variable -> t -> t

mu 'a t is the representation of the recursive type mu 'a. t

val let_ : binding:(variable * t) -> in_:t -> t
